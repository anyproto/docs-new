# Inline Queries

Display a live query or collection inside any Object, right alongside your other content.

Inline queries let you surface a filtered view of your data without leaving the page you're working on. This is useful for dashboards, project pages, or any Object where you want to show a dynamic list of related items.

### Creating an Inline Query

You can create an inline query using the command: `/inline`.

Once created, select an existing query as a source, or create a new one. Any changes you make to the inline query icon or name will be synced with the source set. Meanwhile, views will not be synced but copied, which means you can tweak views to your current needs without worrying that your source query will be impacted.
