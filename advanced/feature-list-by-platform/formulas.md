# Formulas

Perform calculations and aggregations directly in Grid view.

When you're looking at a query or collection in Grid view, you can count objects and run simple math and aggregation functions across any property column. This is useful for things like summing up hours, counting tasks by status, or finding averages.

### Using Formulas

You can count objects in the Grid view and perform simple math and aggregation functions with all types of properties.
